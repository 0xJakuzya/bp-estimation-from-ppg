# BP Estimation from PPG

Проект посвящён оценке систолического и диастолического артериального давления (`SBP`, `DBP`) по PPG-сигналу из датасета PulseDB/VitalDB.

Основная идея проекта — построить классический ML-пайплайн для оценки давления по контактному PPG, проанализировать качество моделей на разных выборках, а затем использовать этот пайплайн как основу для перехода к бесконтактной оценке давления по rPPG/BVP-сигналам, восстановленным из видео с помощью PhysNet.

## Данные

Используются `.mat` файлы PulseDB/VitalDB. Каждый сегмент содержит:

- `Signals` формы `[1250, 3, n_segments]`;
- канал `0` — ECG;
- канал `1` — PPG;
- канал `2` — ABP;
- целевые значения `SBP` и `DBP`;
- метаданные пациента: `Age`, `BMI`, `Gender`, `Height`, `Weight`, `Subject`.

Основной анализ в текущем пайплайне использует только PPG-канал. ABP применяется как источник целевых значений давления.

## Ноутбуки

| Файл | Назначение |
|---|---|
| `01_pulsedb_eda.ipynb` | Разведочный анализ PulseDB: структура данных, распределения SBP/DBP, демография, качество PPG-сигналов, число сегментов на пациента. |
| `02_pulsedb_feature_engineering_classical_ml.ipynb` | Извлечение PPG-признаков и обучение классических моделей: Random Forest, XGBoost, LightGBM, SVR. |

## PPG-признаки

В классическом ML-пайплайне извлекаются признаки нескольких групп:

- статистические признаки исходного PPG: среднее, стандартное отклонение, минимум, максимум, процентили, RMS, skewness, kurtosis;
- признаки первой и второй производной PPG;
- признаки пиков и PRV: число пиков, плотность пиков, IBI, RMSSD, амплитуда, prominence, ширина пиков;
- морфологические признаки пульсовой волны: время подъёма, время спада, длительность импульса, площадь, systolic/diastolic area, half-width, slopes;
- частотные признаки: доминирующая частота, HR-band power, total power, SNR.

## Модели

В ноутбуке `02_pulsedb_feature_engineering_classical_ml.ipynb` сравниваются классические ML-модели:

- Random Forest;
- XGBoost;
- LightGBM;
- SVR с RBF-ядром.

Качество оценивается отдельно для `SBP` и `DBP` с использованием метрик:

- `MAE` — средняя абсолютная ошибка;
- `RMSE` — корень из среднеквадратичной ошибки;
- `ME` — средняя ошибка, показывающая смещение модели.

## Сравнение классических моделей

Результаты из `02_pulsedb_feature_engineering_classical_ml.ipynb`:

| Модель | SBP MAE | DBP MAE | SBP RMSE | DBP RMSE | SBP ME | DBP ME |
|---|---:|---:|---:|---:|---:|---:|
| XGBoost | 11.91 | 7.76 | — | — | — | — |
| LightGBM | 11.22 | 7.27 | 14.53 | 9.37 | -0.05 | 0.02 |
| SVR | 12.68 | 8.25 | 16.43 | 10.66 | -0.52 | -0.24 |
| **Random Forest** | **10.78** | **7.09** | **14.20** | **9.32** | **0.01** | **0.007** |

На полном датасете лучший результат среди моделей показывает `Random Forest`: он даёт минимальные `MAE` и `RMSE` для `SBP` и `DBP`, а также практически нулевое среднее смещение. `LightGBM` остаётся близким по качеству и может рассматриваться как сильная альтернатива.
